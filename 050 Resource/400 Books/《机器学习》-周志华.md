## classification
* predict categories or discrete categories
* small numbers of possible outputs(finite)
### logistic regression
* even though the name of this method is called "regression",but it's a classification method, particularly in binary classfication
#### logistic/sigmoid function
* 逻辑回归的核心是通过一个逻辑函数（通常是Sigmoid函数）将输入特征映射到一个概率值，表示某个事件发生的可能性。
* 数学模型
	* $h(x)=\frac1{1+e^{-(\beta_0+\beta_1x_1+\beta_2x_2+...+\beta_nx_n)}}$
	* β是模型的参数，x是输入特征,由于$h(x)$的值在0-1 之间，因此特别适合表示概率
* 分类
	* 逻辑回归通过设定一个阈值（通常为0.5）来进行分类。如果预测的概率大于这个阈值，则将样本归类为正类（例如1），否则归类为负类（例如0）
#### Decision boundary
* 决策边界（Decision Boundary）是分类模型用于将不同类别的样本分开的边界。它本质上是一种可视化分类效果的方法。
* 对于逻辑回归模型，决策边界是由假设函数的参数决定的（即$z=\beta_0+\beta_1x_1+\beta_2x_2+...+\beta_nx_n$）。

## regression
* predict a number
* infinitely many possible outputs
### linear regression
* with one variable(univariate单变量的) or multiple
#### terminology - notation
* $x=$"input" variable -- feature
* $y=$"output" variable --"target" variable
* $(x,y)=$ single training example
#### cost function(loss function)
* **definition**: Quantifies the difference between *predicted* outcomes and *actual* outcomes
##### MSE(mean squared error),均方误差
* $MSE=(1/m) * Σ(y_{pred} - y_{true})²$
* $J(w,b)=\frac{1}{2m}\sum_{i=1}^{m}\bigl(f_{w,b}\bigl(x^{(i)}\bigr)-y^{(i)}\bigr)^{2}$
#### gradient descent 梯度降阶算法
* **definition**:minimize the cost or loss function of a model
* $w=w-\alpha\frac\partial{\partial w}J(w,b)$
* $\alpha$ -- learning rate
	* how big the step u take downhill
* $\frac\partial{\partial w}J(w,b)$ -- derivative term（导数）
	* which direction u want to take
* normal equation
	* an alternative to gradient descent
##### re-scaling feature

###### mean normalization
* A preprocessing technique applied to datasets to improve the optimization process during model training. It involves *adjusting* the feature values so that they have a mean of zero(平均值为0), which can help enhance the convergence(收敛) speed of gradient descent algorithms.
* $x^\prime=\frac{x-\mu}{\max(x)-\min(x)}$
	* $x\prime$ -- normalized value
	* $x$ -- original value
	* $\mu$ -- the mean of the feature,
###### standardization(z-score normalization)
* $x\prime = \frac{x - \mu}{\sigma}$
	* $x$ -- original feature value
	- $\mu$ -- mean of the feature
	- $\sigma$ -- standard deviation of the feature(标准差)
		- 标准差是方差的平方根
		- $\sigma = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (X_i - \mu)^2}$
	- $x'$ -- the normalized value

##### Batch Gradient Descent
* Each step of gradient descent uses all the training examples

##### checking convergence
* If J（w, b） decreases by ≤ $\epsilon$(0.001) in one iteration，declare convergence.
