* Network Address Translation
* NAT 是一个 网络地址转换 的技术，它通过修改 IP 数据包中的地址信息（源地址或目标地址），使得来自一个网络的流量可以被正确地路由到另一个网络。
* 分类
	* **源地址转换（SNAT）**：修改源 IP 地址，通常用于将私有 IP 地址转换为公有 IP 地址，从而使私有网络中的设备能够访问外部网络（例如访问互联网）。
	    
	- **目的地址转换（DNAT）**：修改目标 IP 地址，通常用于将外部流量的目标 IP 转换为内部网络中的私有 IP 地址。这是 **LVS** 中常见的负载均衡模式之一。
	    
	- **双向 NAT（或称为双向映射）**：同时修改源和目标地址，这种方式较少使用，但在某些特定网络架构中可能会出现。
