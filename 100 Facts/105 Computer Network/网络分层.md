## OSI模型
|层级|层名称|功能描述|
|---|---|---|
|**7**|**应用层**|用户与网络的接口，提供网络服务（如 HTTP、FTP、DNS 等）。|
|**6**|**表示层**|负责数据的格式化、加密与解密、数据压缩等。|
|**5**|**会话层**|管理会话（连接），提供会话管理功能，如同步、恢复等。|
|**4**|**传输层**|提供端到端的通信服务，保证数据的可靠性与流量控制（如 TCP、UDP）。|
|**3**|**网络层**|实现数据的路由与转发，决定数据的路径（如 IP、路由协议）。|
|**2**|**数据链路层**|提供点对点之间的数据传输，负责链路的建立、数据帧的传输等。|
|**1**|**物理层**|负责比特流的物理传输，涉及硬件设备与信号的传输（如电缆、光纤）。|
## TCP/IP模型
| 层级    | 层名称     | 功能描述                                        |
| ----- | ------- | ------------------------------------------- |
| **4** | **应用层** | 负责应用程序之间的通信，提供应用服务（如 HTTP、FTP、SMTP、DNS）。    |
| **3** | **传输层** | 提供端到端的通信服务，控制数据传输的完整性（如 TCP、UDP）。           |
| **2** | **网络层** | 负责数据的路由与传输，控制数据包在网络中的路径（如 IP 协议）。           |
| **1** | **链路层** | 负责在物理网络上发送和接收数据帧，处理硬件接口和数据链路控制（如 Ethernet）。 |
