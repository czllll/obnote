## 1. Background Research for the project

### 1.1 Context

Nowadays, various LLMs possess reasoning capabilities, such as OpenAI's O1 and O3, as well as DeepSeek's R1. Our goal is to explore whether LLMs can play the Taboo game. This study aims to evaluate the performance of LLMs in the Taboo game.

### 1.2 Problem statement

To evaluate whether LLMs can play the Taboo game, we designed the following metrics to explore their capabilities in different aspects:

## As the Clue Giver:

1. **Prompt Effectiveness**: The success rate of guesses based on LLM-generated hints.
2. **Description Diversity**: Whether the LLM can understand the core meaning of the target     word and express the same concept using different linguistic structures.
3. **Taboo Word Violation Rate**: Whether the LLM can recognize and  avoid using the forbidden words.
4. **Clarity of Description**: Evaluating whether the hints are easy to understand.
5. **Adaptive Adjustment Ability**: The capability to adjust the     descriptions dynamically based on context.

As the Guesser:

1. **Semantic Reasoning Ability**: The ability to gradually narrow down the scope of guesses.

### 1.3 Possible solution

We can design two types of experiments to evaluate the above capabilities:

#### **Human vs. LLM**

- **Human as the Clue Giver, LLM as the Guesser**
- **LLM as the Clue Giver, Human as the Guesser**

#### **LLM vs. LLM**

We have designed the following quantitative metrics to evaluate LLM capabilities:

##### **LLM as the Clue Giver**

| **Metric**                      | **Quantification Method**                                    |
| ------------------------------- | ------------------------------------------------------------ |
| **Prompt Effectiveness**        | Number of rounds in which the target word is successfully guessed / Total rounds |
| **Taboo Word Violation Rate**   | Number of violations / Total rounds (lower is better)        |
| **Description Diversity**       | Proportion of different hints (e.g., n-gram diversity)       |
| **Clarity of Description**      | Rated by humans or LLM-as-Judge                              |
| **Adaptive Adjustment Ability** | Evaluates whether the LLM adjusts its strategy over multiple rounds, using BLEU/Rouge to compute similarity between consecutive hints |

##### **LLM as the Guesser**

The following metrics are designed to evaluate the semantic reasoning ability of the guesser:

| **Metric**                                 | **Quantification Method**                                    |
| ------------------------------------------ | ------------------------------------------------------------ |
| **Final Guess Accuracy**                   | Number of successful guesses / Total rounds                  |
| **Average Guess Rounds**                   | Average number of guesses before the correct answer          |
| **Convergence Rate (Semantic Similarity)** | Measures semantic similarity between consecutive guesses using WordNet/BERT cosine similarity |
| **Repetitive Guess Rate**                  | Number of repeated guesses / Total guesses                   |

## 2. Scope for this project

### 2.1 Aim 

The goal of this project is to evaluate the performance of large language models (LLMs) in the Taboo game, analyzing their semantic reasoning ability, adaptability, and ability to follow game rules. This study aims to investigate whether LLMs can effectively take on the roles of both the **Clue Giver** and the **Guesser** and successfully complete the respective tasks:

- **As the Clue Giver**: Can LLMs avoid using taboo words while providing effective hints that enable the guesser to correctly identify the target word?
- **As the Guesser**: Can LLMs reason based on the given hints, progressively narrow down the possible answers, and ultimately identify the correct word?

This project employs controlled experiments combined with quantitative evaluation metrics (e.g., accuracy, number of guessing rounds, error rate) to systematically analyze LLMs’ capabilities in semantic reasoning, language comprehension, interactive inference, and rule adherence. The findings will shed light on the strengths and limitations of LLMs in language-based games and provide valuable insights for future applications of LLMs in natural language interaction and reasoning.

### 2.2 Objectives

* Establishing an Evaluation Framework for LLMs in the Taboo Game

  * Define the roles of LLMs in the game, including Clue Giver and Guesser.

  - Ensure that the evaluation framework is standardized and supports quantifiable experimental results.

* Comparing the Performance of Different LLMs in the Taboo Game
  - Test various LLMs to evaluate their reasoning ability and language generation quality.

* Evaluating LLMs’ Rule Compliance and Adaptability

  - As the **Clue Giver**, can LLMs avoid using taboo words while providing effective hints?

  - As the **Guesser**, can LLMs adjust guessing strategies effectively, avoiding random or repetitive guesses?

## 3. Project schedule

### 3.1 Methodology

* **Experiment Design**

  * define LLM roles
    - assign LLMs as clue givers and guessers.

  * target and taboo word generation

    - generate **target words** based on word frequency, part of speech（noun, verb, adjective）, and complexity.

    - select taboo words using word embeddings or lexical databases.

  * experimental variables

    - test different LLMs.

    - vary the number of taboo words.

  * evaluation metrics

    - **success rate**: correctly guessed target words.

    - **error rate**: taboo word violations.

    - **semantic similarity:** alignment of hints with the target word.

    - **average number of guesses**: efficiency in finding the correct answer.

* **Web Platform Development**

  - Use Next.js + Tailwind CSS to build the front end, providing an intuitive experimental interface.

  - Use FastAPI / Flask to develop the back end, handling LLM interactions and data storage.

  - Allow users to select LLM models, configure experimental parameters, and run experiments.

* **Experiment Execution**

  - Conduct experiments and collect interaction data from LLMs under different conditions.

  - Record each round’s semantic similarity, error rate, and success rate for further analysis.

* **Data Analysis**

  - Use Python data analysis libraries (Pandas, Matplotlib, SciPy) for statistical analysis.

  - Generate visual representations such as line charts and bar graphs.

  - Compare different LLMs in terms of rule compliance, reasoning paths, and error rates.

* **Report Writing**

  - Summarize experimental results and write the research paper.

  - Discuss LLM performance in the Taboo game, analyzing strengths, limitations, and potential improvements.
