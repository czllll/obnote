* Warehouse Scale Computing(WSC)
	* HPC(High Performance Computing)
	* Design of it
		* Cost-performance
		* Energy efficiency
		* Dependability via redundancy
		* Network IO
		* workloads
	* Different between HPC and WSC
		* HPC optimizes for computation speed while WSC optimizes for data processing scale and cost efficiency.
* Heterogeneity异构性
	* Heterogeneity refers to the diverse mix of hardware, architectures and performance characteristics across computing systems.
	* Macro level宏观
		* networks
		* composed by diverse node architectures
		* interconnected with potentially heterogeneous networks
	* Micro level
		* deeper memory hierarchies(main, cache....)
		* accelerator architectures(fixed, programmable, eg. GPUs)
	* Other :
		* Soft ware, middleware, tools.
* hardware Classification
	* GPP(General Purpose Processors)
	* ASP(Application Specific Processors)
		* DSP Digital Signal Processor
		* ASIP Application Specific Instruction Set Processors 应用特定指令集处理器
		* ASIC Application Specific Integrated Circuit 应用特定集成电路
	* Accelerators-Coprocessors
		* GPU
		* FPGA Programmable Gate Array
			* FPGA is an array of logic gates that can be hardware-programmed to fulfill use-specified tasks

* Storage and Array Switch
	* SAN Storage Area Network
		* local network that uses Fibre Channel(光纤) to connect several data storage devices
	* NAS Network Attached Storage
		* dedicated hardware device that connects to a local area network, usually through an Ethernet connection and operates on data files
