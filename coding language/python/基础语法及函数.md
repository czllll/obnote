### 生成器
* 当函数内部使用`yield`关键字时，该函数被称为生成器函数；当调用生成器函数时，返回的是一个生成器对象（即迭代器）
* 用于创建迭代器，返回一系列的值，每次调用生成一个值（惰性计算：节省内存）
* 返回使用`yield`而非`return`,每次遇到`yield`函数暂停并返回一个值
```python
def count_up_to(max_value): 
	count = 1 
	while count <= max_value: 
		yield count 
		count += 1
```
### 纯函数
#### 纯函数
* 特点
	* 无副作用：不改变外部状态
	* 相同输入必有相同输出
#### 非纯函数
* 特点
	* 有副作用
	* 输出不稳定：输出不仅仅以来输入参数，还可能依赖外部状态。相同的输入在不同环境下可能有不同的返回结果
### 闭包（Closure）
#### 是什么
* 闭包由外部函数、内部函数、外部函数的局部变量组成
* 当inner引用outer的局部变量时，即使outer已经执行完毕，inner仍可以访问那些变量
#### 基本结构
```python
def outer_function(msg):
    def inner_function():
        print(msg)
    return inner_function
closure = outer_function("Hello, World!")
closure()  # 输出: Hello, World!
```
* **注意**：outer返回inner，相当于inner捕获了outer的局部变量，即使outer早就执行完了，也可以继续使用那些变量
#### nonlocal
* 在闭包的上下文中，`nonlocal`关键字允许内部函数修改外部函数的局部变量，以实现状态的更新或持久化。

## 语法
### 推导式语法
#### 列表推导式
* 用于生成新的列表
* `[表达式 for 变量 in 可迭代对象 if 条件]`
```python
# 生成一个包含1到10之间所有偶数的列表 
evens = [x for x in range(1, 11) if x % 2 == 0]
```
#### 集合推导式
* 用于生成集合，使用{}
* `{表达式 for 变量 in 可迭代对象 if 条件}`
```python
# 生成一个包含1到10之间所有偶数的集合 
evens_set = {x for x in range(1, 11) if x % 2 == 0}
```
#### 字典推导式
* 用于生成字典
* `{键表达式: 值表达式 for 变量 in 可迭代对象 if 条件}`
```python
# 生成一个字典，键是1到5，值是键的平方 
squares = {x: x**2 for x in range(1, 6)}
```
#### 生成器表达式
* 返回生成器对象，使用小括号()
```python
# 创建一个生成器对象，生成1到10之间所有偶数 

evens_gen = (x for x in range(1, 11) if x % 2 == 0)
```

## 对象
### 迭代器
#### 是什么
* 是一个对象，实现了两个方法
	* `__iter__()`
	* `__next__()`
* 迭代器可以逐个访问序列中的元素，而无序将所有元素同时加载到内存中
#### 可迭代对象是什么
* 是指实现了`__iter__()`方法的对象，可迭代对象可以生成一个迭代器，用于逐步访问其元素。eg：列表、元组、字典等
#### 如何创建迭代器
* 使用生成器
* 自定义
#### `iter()`函数是做什么用的
* `iter()`接受一个可迭代对象，并返回一个迭代器
* 使用`iter()`,可只在需要时获取下一个元素。对比切片而言，无需创建一个新的列表（浪费内存）
#### 适用场景
* 大数据和无限数据流
* 需要惰性计算时
