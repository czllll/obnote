# 题面

有一个只含有 `'Q', 'W', 'E', 'R'` 四种字符，且长度为 `n` 的字符串。

假如在该字符串中，这四个字符都恰好出现 `n/4` 次，那么它就是一个「平衡字符串」。

 

给你一个这样的字符串 `s`，请通过「替换一个子串」的方式，使原字符串 `s` 变成一个「平衡字符串」。

你可以用和「待替换子串」长度相同的 **任何** 其他字符串来完成替换。

请返回待替换子串的最小可能长度。

如果原字符串自身就是一个平衡字符串，则返回 `0`。

 

**示例 1：**

```
输入：s = "QWER"
输出：0
解释：s 已经是平衡的了。
```

**示例 2：**

```
输入：s = "QQWE"
输出：1
解释：我们需要把一个 'Q' 替换成 'R'，这样得到的 "RQWE" (或 "QRWE") 是平衡的。
```

**示例 3：**

```
输入：s = "QQQW"
输出：2
解释：我们可以把前面的 "QQ" 替换成 "ER"。 
```

**示例 4：**

```
输入：s = "QQQQ"
输出：3
解释：我们可以替换后 3 个 'Q'，使 s = "QWER"。
```

# 题解
## 题意

* 字符串s平衡说明每个字母都占相同的长度即length = len(s) // 4，即寻找使得在替换子串之外的元素的个数均不超过length，则说明可以通过替换来使得题目成立

## 思路

* 经典变长滑动窗口，窗口即为替换子串，首先统计所有字符的个数为初始窗口外的计数cnt（因为初始窗口大小为0），枚举右端点right，枚举到的字符计数减一，即说明该字符进入窗口。
* while循环判断当前窗口外的字符计数是否均不超过length，若成立则更新答案ans，并向右移动左端点left

```python
class Solution:
    def balancedString(self, s: str) -> int:
        cnt = defaultdict(int)
        for i in s:
            cnt[i] += 1
        left = 0
        ans = inf
        length = len(s) // 4
        if len(cnt) == 4 and min(cnt.values()) == length:
            return 0
        for right, num in enumerate(s):
            cnt[num] -= 1
            while max(cnt.values()) <= length:
                ans = min(ans, right - left + 1)
                cnt[s[left]] += 1
                left += 1
        return ans
```



## 复杂度
* 时间
  * O(nC) C为4，计算max为4
* 空间
  * O(C)，哈希表存储
