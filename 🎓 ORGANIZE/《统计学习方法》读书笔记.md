# 概论
## 基本分类
### **监督学习**
* 从*标注数据*中学习预测模型的机器学习问题
* *标注数据*：输入输出的对应关系
* 本质：学习输入到输出的映射统计规律
#### 概念
* **输入空间、特征空间、输出空间**
	* **输入/出空间**：输入与输出所有可能取值的集合。
		* 可以是有限元素集合，也可以是整个[[欧氏空间]]
		* 可以是相同或不同空间， 通常输出空间远小于输入空间
	* **特征空间**：每一个具体输入是一个实例，通常用特征向量表示，所有特征向量存在的空间叫特征空间。
	* 输入变量与输出变量均为连续变量的预测问题称为**回归**问题；输出变量为有限个离散变量的预测问题称为**分类**问题；输入变量与输出变量均为变量序列的问题称为**标注**问题。
* **联合概率分布**
	* 监督学习**假设**输入与输出的随机变量X和Y遵循联合概率分布P(X,Y)
		* $P(X,Y)$为[[分布密度函数]]
	* 训练数据和测试数据被看作是依$P(X,Y)$[[独立同分布]]产生的。
* **假设空间**
	* 输入空间到输出空间所有映射（由模型表示）的集合。
	* 监督学习模型可以是
		* 概率模型 $P(Y|X)$
		* 非概率模型$Y=f(x)$
* **问题形式化**
	* 监督学习分为学习、预测两个过程。分别由学习系统和预测系统完成
### 无监督学习
* 从无标注数据中学习预测模型的机器学习问题
* 本质是学习数据中的统计规律或潜在结构
#### 概念
* 输出/出空间同上
	* 每一个输出是对输入的分析结果，由输入的类别、转换或概率表示。
* 结构空间(S)：描述数据组织方式或内在关系的空间
* 模型可以实现对数据的聚类、降维或概率估计
* 包含所有可能模型的集合称为假设空间，无监督学习旨在从假设空间中选择给定评价标准下的最佳模型
* 同样也分为学习和预测两个系统完成

### 强化学习
* 智能系统在与环境的连续互动中学习最优行为策略的问题
	* 假设智能系统与环境的互动基于[[马尔可夫决策过程]]，智能系统能观测到的就是与环境互动得到的数据序列。
	* 强化学习的本质是学习最优的序贯序列
* 智能系统与环境的互动，在每一步t
	* 智能系统
		* 从环境观测到一个state($s_t$)，与一个reward($r_t$)
		* 采取一个action($a_t$)
	* 环境
		* 根据智能系统选择的动作决定下一步的state($s_{t+1}$)，reward($r_{t+1}$)
* 要学习的策略是
	* 给定state下采取的action
* 智能系统的目标不是短期reward最大化，而是长期累积的reward最大化。强化学习过程中通过不断的trail and error，以达到最优学习策略
#### 强化学习的马尔可夫决策过程
* 强化学习的马尔可夫决策过程是state, reward, action序列上的随机过程，由五元组$<S,A,P,r,\gamma>$组成
* S有限状态机和
* A有限动作集合
* P状态转移概率函数
* r奖励函数
	* $r(s,a) = E(r_{t+1}|s_t = s, a_t = a)$
	* 在当前状态 s 下采取动作 a，预期能够获得的下一步奖励的平均值
* 策略$\pi$是给定状态下动作$a = f(s)$或者条件概率分布$P(a|s)$
* 状态价值函数为在策略$\pi$从某一个state开始的长期累积reward的数学期望
	* $v_\pi(s) = E_\pi[r_{t+1} + \gamma r_{t+2} + \gamma^2 r_{t+3} + \cdots|s_t = s]$
* 动作价值函数定义为策略$\pi$从某一个state和action开始的长期累积reward的数学期望
	* $q_\pi(s,a) = E_\pi[r_{t+1} + \gamma r_{t+2} + \gamma^2 r_{t+3} + \cdots|s_t = s, a_t = a]$

### 半监督学习和主动学习
* 半监督
	* 通常有少量标注数据，大量未标注数据
	* 旨在利用未标注数据中的信息，辅助标注数据进行监督学习
* 主动
	* 目标是找出对学习最有帮助的实例让教师来标注
* 以上两种更接近于监督学习
## 按模型分类
## 概率模型与非概率模型
* 条件概率分布和函数可以互相转化。
	* 条件概率分布最大化后得到函数
	* 函数归一化后得到条件概率分布
* 概率模型和非概率模型之间不存在
