### Distributed Computing Paradigms
* Internet, TCP/IP
	* Link add machines together
* WWW,HTTP,HTML,XML
	* Link all documents together
* Web services, SOAP, WSDL, REST
	* Link all application together
* Decentralised/distributed infrastructures
	* the grid
	* the cloud
### Definition of **Cloud computing**
* Cloud computing is a model for enabling ubiquitous, convenient, on-demand network access to a shared pool of configurable computing resources(eg, networks, servers, storage, app and services) that can be rapidly provisioned and released with minimal management effort or service provider interaction. (NIST)
	* On-demand self-service
	* Broad network access
	* Resource pooling
	* Rapid elasticity or expansion
	* Measured service
* 3 *service models*
	* software
	* platform
	* infrastructure
		* key technology: **virtualization**
* 4 *deployment models*
	* private
	* community
	* public
	* hybrid
