| 维度             | 指标                           | 公式 / 说明                                                                       |
| -------------- | ---------------------------- | ----------------------------------------------------------------------------- |
| **约束满足**       | **Violation Rate**           | 是否触碰禁忌词；二元计数                                                                  |
|                | **Format Pass Rate**         | 正则匹配通过率                                                                       |
| **提示信息效率**     | **Info-Density Qhint**       | $\frac{\text{Sim}(hint, target)}{\text{tokens}(hint)}$Sim = cosine(embedding) |

|                | Avg tokenConsumption@Success | 成功样本平均轮数平均token消耗                                                             |
|                | **Failure Mode**             | 未猜中、禁忌违规等占比                                                                   |
