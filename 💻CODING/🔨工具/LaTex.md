# 基础命令
* `\documentclass[UTF8]{ctextart/article/reporter}`
	* `\begin{document}`
	* `\end{document}`
* 
* `\textbf{}`
* `\textit{}`
* `\underline{}`
* 两个换行符-- 新一段
* `\part{}`
* `\chapter{}`
	* `section{}`--章节
		* `\subsection{}`
* 添加图片
	* 在前言添加`\usepackage{graphicx}`
	* `\includegraphics[width=0.5\textwidth]{image}`
	* 添加图片标题
		* `\begin{figure}`
		* `\centering`
		*  `\includegraphics[width=0.5\textwidth]{image}`
		* `\caption{title}`
		* `\end{figure}`
* 公式
```latex
\frac //分数
\sum //求和符号
_ // subscript
^ // superscirpt
\bigl(
\bigr) //括号
```
