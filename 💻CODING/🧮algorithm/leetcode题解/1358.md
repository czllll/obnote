# 题面

给你一个字符串 `s` ，它只包含三种字符 a, b 和 c 。

请你返回 a，b 和 c 都 **至少** 出现过一次的子字符串数目。

 

**示例 1：**

```
输入：s = "abcabc"
输出：10
解释：包含 a，b 和 c 各至少一次的子字符串为 "abc", "abca", "abcab", "abcabc", "bca", "bcab", "bcabc", "cab", "cabc" 和 "abc" (相同字符串算多次)。
```

**示例 2：**

```
输入：s = "aaacb"
输出：3
解释：包含 a，b 和 c 各至少一次的子字符串为 "aaacb", "aacb" 和 "acb" 。
```

**示例 3：**

```
输入：s = "abc"
输出：1
```

## 思路

* 判断字典长度为三时，当前窗口对应的子串成立，则以当前窗口为开头子串的字符串均成立，因此 `ans += len(s) - right`

```python
class Solution:
    def numberOfSubstrings(self, s: str) -> int:
        ans = left = 0
        cnt = defaultdict(int)
        cnt_s = Counter(s)
        for right, c in enumerate(s):
            cnt[c] += 1
            while len(cnt) == 3:
                ans += len(s) - right
                cnt[s[left]] -= 1
                if cnt[s[left]] == 0:
                    del cnt[s[left]]
                left += 1
        return ans
```



## 复杂度
* 时间
  * O(n)
* 空间
  * O(1)
