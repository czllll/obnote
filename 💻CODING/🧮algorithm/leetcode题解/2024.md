[2024. 考试的最大困扰度 - 力扣（LeetCode）](https://leetcode.cn/problems/maximize-the-confusion-of-an-exam/description/)
## 题面
A teacher is writing a test with `n` true/false questions, with `'T'` denoting true and `'F'` denoting false. He wants to confuse the students by **maximizing** the number of **consecutive** questions with the **same** answer (multiple trues or multiple falses in a row).
You are given a string `answerKey`, where `answerKey[i]` is the original answer to the `ith` question. In addition, you are given an integer `k`, the maximum number of times you may perform the following operation:
- Change the answer key for any question to `'T'` or `'F'` (i.e., set `answerKey[i]` to `'T'` or `'F'`).
Return _the **maximum** number of consecutive_ `'T'`s or `'F'`s _in the answer key after performing the operation at most_ `k` _times_.

**Example 1:**

**Input:** answerKey = "TTFF", k = 2
**Output:** 4
**Explanation:** We can replace both the 'F's with 'T's to make answerKey = "TTTT".
There are four consecutive 'T's.

**Example 2:**

**Input:** answerKey = "TFFT", k = 1
**Output:** 3
**Explanation:** We can replace the first 'T' with an 'F' to make answerKey = "FFFT".
Alternatively, we can replace the second 'T' with an 'F' to make answerKey = "TFFF".
In both cases, there are three consecutive 'F's.
## 题解
### 题意
* 实际上是求$answerKey$的一个最长子串,至多包含k个T**或**k个F
### 思路
* 这道题同样是长度变化的滑动窗口。
* 遍历$answerKey$，枚举右端点$right$，同时维护最小左端点left和字串中的字符个数$cnt$。
* 把$answerKey[right]$的出现次数加一
* 若窗口内的T与F数量均超过k，则说明窗口目前无法满足全部为连续出现的$T$或$F$。那么必须不断移动左端点$left$，同时减少$answerKey[left]$的出现次数，直到满足$T$和$F$的出现次数至少有一个$<=k$.
* 循环结束后，$right-left+1$即为窗口长度，更新最大$ans$。
* 直至$right$枚举完整个数组。

```python
class Solution:
    def maxConsecutiveAnswers(self, answerKey: str, k: int) -> int:
        ans = 0
        left = 0
        cnt = defaultdict(int)
        for right, ch in enumerate(answerKey):
            cnt[ch] += 1
            while cnt["T"] > k and cnt["F"] > k:
                cnt[answerKey[left]] -= 1
                left += 1
            ans = max(ans, right - left + 1)

        return ans
```
### 复杂度
* 时间复杂度：
	* O(n)，n为answer的长度
	* 内层while循环，实际上对left加一的次数不会超过n，因此为常数级时间复杂度
* 空间复杂度：
	* 使用了常数级的空间
