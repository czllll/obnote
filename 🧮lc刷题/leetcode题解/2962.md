# 题面

给你一个整数数组 `nums` 和一个 **正整数** `k` 。

请你统计有多少满足 「 `nums` 中的 **最大** 元素」至少出现 `k` 次的子数组，并返回满足这一条件的子数组的数目。

子数组是数组中的一个连续元素序列。

 

**示例 1：**

```
输入：nums = [1,3,2,3,3], k = 2
输出：6
解释：包含元素 3 至少 2 次的子数组为：[1,3,2,3]、[1,3,2,3,3]、[3,2,3]、[3,2,3,3]、[2,3,3] 和 [3,3] 。
```

**示例 2：**

```
输入：nums = [1,4,2,1], k = 3
输出：0
解释：没有子数组包含元素 4 至少 3 次。
```

## 思路

* 可变滑动窗口，更新ans时要考虑当前满足情况可以增加多少个答案

```python
class Solution:
    def countSubarrays(self, nums: List[int], k: int) -> int:
        left = ans = 0
        max_v = max(nums)
        cnt_max = 0
        for right in range(len(nums)):
            if nums[right] == max_v:
                cnt_max += 1
            while cnt_max == k:
              # ans += len(s) - right ans在循环内更新
                if nums[left] == max_v:
                    cnt_max -= 1
                left += 1
            ans += left #ans在循环外更新

        return ans
```

* 在while循环内部更新即，当前结果的情况为最小情况，以当前窗口为透子串的字符串均满足题意，因此直接更新增加余下的字符个数
* 而在while循环外部更新，当前的cnt_max一定是小于k的，那么对于右端点为 *right* 且左端点小于 *left* 的子数组，肯定全部满足题意

## 复杂度
* 时间
  * O(n)
* 空间
  * O(1)
