#### 2. Arrays.sort()底层实现？#javase

* 总体上`Arrays.sort()`方法对基本数据类型和包装类型采取不同的排序策略。
* JDK6及更早，Java对基本数据类型使用经典单轴快排，对对象数组使用的是归并排序
* JDK7对基本数据类型引入了双轴快排，对对象数组使用的是TimSort（归并 + 插排）
* JDK14对sort进行了重构，对具体的排序策略进行了一定优化

#### 4. MySQL索引失效情况？#MySQL

* %模糊查询前置通配符
* Where中or某个分支没有使用索引，在优化器评估成本后，可能放弃索引使用
* 查询对索引列计算或使用函数，类型不一致导致的隐式转换，相当于使用了函数
* 联合索引没有遵循最左匹配
* 排序方向不一致导致索引失败，联合索引只支持**同方向排序**

#### 5. 如何知道走索引？ #MySQL

* 使用explain命令执行查询，主要观察possible key和type, extra
* Type: const, eq_ref, ref, range, index ,all
* Extra:
  * using index : 使用了覆盖索引
  * using filesort表示mysql需要额外排序，没有用到索引排序
  * using temporary: 表示用到了临时表，多见于group by

#### 6. Spring 如何统计某个service 层方法被调用了多少次？#Spring

* 使用aop，建立一个切面类
  * 定义切点Pointcut
    * 拦截哪些方法
  * 编写通知Advice
    * 前置通知（Before）
    * 后置通知（After/AfterReturning）
    * 环绕通知（Around）
    * 异常通知（AfterThrowing）
  * JoinPoint 拿方法的上下文，作为advice方法的参数
  * 注册切面Aspect
    * 使用@Aspect @Component 标记
* 可以在定义一个注解类，改一下Pointcut拦截就行了
